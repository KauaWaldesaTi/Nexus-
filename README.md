## 🚀 Sobre o Projeto

O **Nexus** é um sistema completo de monitoramento e gestão de equipamentos voltado para a **ASTEC Drives**, desenvolvido para acompanhar o desempenho, status e funcionamento de ativos em tempo real.

A plataforma centraliza informações críticas, permitindo maior controle operacional, redução de falhas e tomada de decisão baseada em dados.

## 🎯 Objetivo

Desenvolver uma solução robusta para:

- Monitoramento de equipamentos
- Gestão de ativos
- Visualização de status em tempo real
- Organização de dados técnicos
- Controle operacional centralizado

- Bootstrap

## ⚙️ Funcionalidades

- 📊 Dashboard de monitoramento em tempo real

## 🏗️ Arquitetura

O sistema foi desenvolvido com foco em escalabilidade e organização:

- Front-end com Blazor + Bootstrap
- Back-end em C#
- Integração com banco de dados relacional
- Separação por camadas

---

## 🗄️ Banco de Dados

O projeto conta com uma estrutura robusta, contendo **mais de 30 tabelas**, garantindo organização e alta capacidade de armazenamento.

### Principais entidades:

- Equipamentos
- Sensores
- Leituras
- Usuários
- Logs
- Alertas
- Histórico de eventos
